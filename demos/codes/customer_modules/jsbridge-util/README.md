### 介绍
JSBridge的上层封住，实现了cache、net、forward、driver等服务,必须依赖于jsbridge-core运行

### GIT
http://10.66.72.20:8081/mobile-front-end/jsbridge-util.git

### 引入及使用

#### 1.安装
    npm install --save jsbridge-util

#### 2.页面引入
